# 패치 노트
## 2020-05-01. ver 1.4.0
- Convex Mesh를 사용하는 Mesh Collider 컴포넌트 추가
## 2020-05-01. ver 1.3.0
- 게임 엔진 안정성 증가
- Cursor 클래스 추가
  - Cursor 클래스는 커서의 보이기 상태, 제한 상태 등을 설정하거나 가져오는 정적 속성을 제공합니다.
- CharacterController 컴포넌트 추가
  - CharacterController 컴포넌트는 게임 개체가 물리 개체와 상호작용하면서 캐릭터 제어를 수행할 수 있도록 합니다.
## 2020-04-22. ver 1.2.3
- Application.OnActive 가상 함수를 추가하였습니다. 사용자는 이 함수를 재정의하여 앱이 활성화될 때, 비활성화될 때의 행동을 정의할 수 있습니다.
## 2020-04-21. ver 1.1.2

- 콜라이더 개체들의 속성이 로컬 좌표를 기준으로 하도록 변경되었습니다.
